🌟 تعلم تركيب الكلمات

في هذا الدرس سنتعلم كيف نجمع الحروف لنكوّن كلمات مفيدة، ونقرأها بطريقة صحيحة وممتعة.

ب + ا + ب

نركب الحروف معاً لنحصل على كلمة:

باب

🔊 نقرأ: باب

ق + ل + م

نركب الحروف معاً لنحصل على كلمة:

قلم

🔊 نقرأ: قلم

ك + ت + ا + ب

نركب الحروف معاً لنحصل على كلمة:

كتاب

🔊 نقرأ: كتاب

ب + ي + ت

نركب الحروف معاً لنحصل على كلمة:

بيت

🔊 نقرأ: بيت

🎯 تمرين

ركب الحروف لتكوين كلمات:

م + د + ر + س + ة = ؟؟

ش + م + س = ؟؟

و + ر + د + ة = ؟؟

حاول قراءة الكلمات بعد تركيبها.

💡 معلومة

عندما نجمع الحروف بالترتيب الصحيح نحصل على كلمات لها معنى.
مثال:
ب + ا + ب = باب
ك + ت + ا + ب = كتاب

⭐ نشاط إضافي

صل الحروف بالكلمة المناسبة:

ب + ح + ر ⟵ بحر

ش + ج + ر + ة ⟵ شجرة

ق + م + ر ⟵ قمر
