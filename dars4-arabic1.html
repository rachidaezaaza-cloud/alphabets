🌟 تعلم تركيب الكلمات

في هذا الدرس سنتعلم كيف نجمع الحروف لنكوّن كلمات مفيدة ونقرأها بطريقة ممتعة.

ب + ا + ب

نركب الحروف لنحصل على:

باب

ق + ل + م

نركب الحروف لنحصل على:

قلم

ك + ت + ا + ب

نركب الحروف لنحصل على:

كتاب

ب + ي + ت

نركب الحروف لنحصل على:

بيت

🎯 تمرين

ركب الحروف لتكوين كلمات:

م + د + ر + س + ة = ؟

ش + م + س = ؟

و + ر + د + ة = ؟

حاول قراءة الكلمات بصوت واضح.

💡 معلومة

عندما نرتب الحروف بشكل صحيح نحصل على كلمة لها معنى.

مثال:
ب + ا + ب = باب
ك + ت + ا + ب = كتاب

⭐ نشاط إضافي

صل الحروف بالكلمة المناسبة:

ب + ح + ر ⟵ بحر

ش + ج + ر + ة ⟵ شجرة

ق + م + ر ⟵ قمر
